
CRIAR PROJETO BACKEND COM ADONIS

    adonis new nomeDiretorio --api-only (--api-only == não tras a parte das views)

CRIAR AS MIGRATIONS -> adonis make:migration type_user
INICIAR AS MIGRATIONS -> adonis migration:run
VOLTAR A AÇÃO -> adonis migration:reset
VER AS TABELAS CRIADAS -> adonis migration:status


POPULAR TABELAS
CRIAR SEED -> adonis make:seed typeUser   
RODAR AS SEEDS -> adonis seed

CRIAR MODEL -> adonis make:model typeUser
CRIAR CONTROLLER -> adonis make:controller user

VER AS ROTAS EXISTENTES -> adonis route:list
